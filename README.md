# Quark Plugin Pack

两个独立插件，可单独安装或一起使用。

---

## 安装

将 zip 直接传入 AstrBot 的插件管理页面上传即可。

---

## 1. quark_qq_inject — QQ 号注入插件

在消息头部插入 `[QQ:xxxxx]`，使 AI 在上下文中能看到发送者的 QQ 号。

- 自动提取字段：`user_id` → `qq` → `uin`
- 避免重复注入
- 静默运行，不产生任何回复

---

## 2. quark_daily_sign — Quark 随机工具箱

一个插件搞定所有随机需求：抽签、运势、骰子、抛硬币、做决定。

### 抽签

发送以 **X签** 结尾的消息即可抽取对应签（签必须在末尾）：

```
来根饭签
抽个卷签
今日睡签
```

发送 **抽签** 二字随机出一支签。

发送 **抽签列表** 查看所有支持的签。

**16 种签主题：** 饭签 · 卷签 · 躺签 · 学签 · 工签 · 摸签 · 氪签 · 水签 · 乐签 · 宅签 · 练签 · 混签 · 旅签 · 睡签 · 发签 · 考签

输出示例：

```
📜 Quark饭签
🏷️ 签文：厨神附体签
⭐ 签面：大吉
✅ 宜：点杯奶茶 · 好好吃饭 · 细嚼慢咽
❌ 忌：空腹硬撑 · 暴饮暴食 · 深夜吃撑
💬 Quark批注：今天你也是米其林三星预备役。
```

每人每天每组每签只能抽一次。

### 今日运势

发送 **今日运势** 查看当天运势。

- 等级：大雄 → 大凶 → 小凶 → 平 → 小吉 → 中吉 → 大吉 → 超级吉 → 妲己
- 每个等级有不同的宜/忌数量和内容
- 大吉以上：诸事皆宜
- 宜忌贴近学生生活（写作业、打游戏、敲代码、上号、翘课、桌游……）
- 每人每天一组一次

```
🔮 今日运势
📊 运势：大吉
🏆 好运连连
✅ 宜：写作业 · 打游戏 · 社交 · 看书
❌ 忌：✨ 诸事皆宜
💬 箴言：好运气藏在你的每一个选择里。
```

### 骰子

发送含 **骰子／色子／dice／roll** 的消息：

```
🎲 你掷出了 4 ⚃
```

### 抛硬币

发送含 **抛硬币／硬币／掷硬币** 的消息：

```
🪙 正面！
```

### 做决定

发送含 **做决定** 的消息，或发送含有 **要不要** + **夸克／quark** 的消息：

```
🤔 让我想想……
➡️ 冲！
```

支持 16 种随机回答（冲、别去、必须的、算了吧……）。

---

### 数据说明

插件运行时会在自身目录下创建 `data/` 文件夹，存放每日缓存：

- `daily_sign_cache.json` — 抽签记录
- `fortune_cache.json` — 运势记录

卸载插件前可删除这两个文件。

### 打包

运行项目根目录的 `pack_zips.py` 即可重新打包两个插件：

```
python pack_zips.py
```
