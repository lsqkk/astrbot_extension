# Quark Plugin Pack

两个插件。

---

## 1. quark_qq_inject — QQ 号注入插件

**功能**：拦截所有消息，将发送者 QQ 号注入消息文本头部 `[QQ:xxxxx]`，  
使 AI 在上下文中能看到 QQ 号；同时注入 `event.injected_qq` 供插件使用。

**特性**：
- 静默运行，不产生任何回复
- 自动回退字段：`user_id` → `qq` → `uin` → `get_sender_name()`
- 避免重复注入（只会添加一次）
- 抽签插件会自动清理 `[QQ:xxx]` 前缀再提取签主题

---

## 2. quark_daily_sign — Quark 每日签

**功能**：抽签插件，支持 20 种签主题，每人每天每组限抽一签。

### 使用方式
发送消息以 **X签** 结尾即可抽签（签必须在末尾）：
```
今天什么饭签
来根卷签
抽个躺签
```
✅ 支持：`来根饭签` `抽个学习签` `今日睡签`
❌ 不支持：`签来一个` `签文`（签不在末尾）

| 品牌名 | 云团子/小云 | Quark |

| 主题数 | 12 | 20 |
| 每条签文 | 4名/2宜/2忌/1注 | 6名/5宜/5忌/2注 |

| QQ来源 | 自提取 | 优先使用注入插件 |
