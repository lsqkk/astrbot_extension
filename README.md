## 安装

### 方式一：上传 zip（推荐）

1. 运行 `python pack_zips.py` 在项目根目录生成 `.zip` 包
2. 打开 AstrBot 管理后台 → **插件管理**
3. 点击 **上传插件**，选择对应 zip 文件即可

### 方式二：目录部署

将 `quark_qq_inject/` 或 `quark_daily_sign/` 整个目录复制到 AstrBot 的 `plugins/` 目录下，重启 AstrBot 即可。

> 两个插件相互独立，可按需只安装其中一个。

## 插件列表

### 1. Quark QQ 注入

**静默注入 QQ 号，让 AI 看到是谁在说话。**

在每条消息头部插入 `[QQ:xxxxx]` 标记，使上游 AI 模型在上下文中感知发送者的身份。

- 自动提取字段：`user_id` → `qq` → `uin`
- 已注入的消息不会重复标记
- 纯静默运行，不产生任何回复消息

**适用场景：** 需要 AI 区分不同用户的群聊、需要基于用户身份做个性化回复的场景。

### 2. Quark 随机工具箱

**集抽签、运势、骰子、抛硬币、做决定于一体的娱乐插件。**

#### 抽签

发送以 **X签** 结尾的消息（签必须在末尾）：

```
来根饭签
抽个卷签
今日睡签
```

发送 **抽签** 二字随机出签；发送 **抽签列表** 查看所有支持的签种。

**16 种签主题：**

饭签 · 卷签 · 躺签 · 学签 · 工签 · 摸签 · 氪签 · 水签 · 乐签 · 宅签 · 练签 · 混签 · 旅签 · 睡签 · 发签 · 考签

输出示例：

```
📜 Quark饭签
🏷️ 签文：厨神附体签
⭐ 签面：大吉
✅ 宜：点杯奶茶 · 好好吃饭 · 细嚼慢咽
❌ 忌：空腹硬撑 · 暴饮暴食 · 深夜吃撑
💬 Quark批注：今天你也是米其林三星预备役。
```

> 每人每天每组每签只可抽取一次。

#### 今日运势

发送 **今日运势** 获取当日运势解读。

- **等级体系：** 大雄 → 大凶 → 小凶 → 平 → 小吉 → 中吉 → 大吉 → 超级吉 → 妲己
- 运势越高，宜项越多，忌项越少
- 大吉及以上等级：诸事皆宜
- 宜忌内容贴近学生与日常生活

```
🔮 今日运势
📊 运势：大吉
🏆 好运连连
✅ 宜：写作业 · 打游戏 · 社交 · 看书
❌ 忌：✨ 诸事皆宜
💬 箴言：好运气藏在你的每一个选择里。
```

> 每人每天可查看一次。

#### 骰子

发送含 **骰子／色子／dice／roll** 的消息：

```
🎲 你掷出了 4 ⚃
```

#### 抛硬币

发送含 **抛硬币／硬币／掷硬币** 的消息：

```
🪙 正面！
```

#### 做决定

发送 **做决定**，或发送含有 **要不要** + **夸克／quark** 的消息：

```
🤔 让我想想……
➡️ 冲！
```

内置 16 种随机回答，涵盖积极、保守、整蛊等多种风格。

## 数据说明

插件运行时会在自身目录下创建 `data/` 文件夹用于持久化缓存：

| 文件 | 说明 |
|------|------|
| `daily_sign_cache.json` | 抽签记录（按用户 × 日期 × 签种） |
| `fortune_cache.json` | 运势记录（按用户 × 日期） |

卸载插件前如有需要可手动删除这两个文件。

## 自行打包

项目根目录提供了打包脚本，一键生成两个插件的 `.zip` 文件：

```bash
python pack_zips.py
```

输出文件：

```
quark_qq_inject.zip
quark_daily_sign.zip
```

将 zip 直接传入 AstrBot 插件管理页面上传即可使用。
